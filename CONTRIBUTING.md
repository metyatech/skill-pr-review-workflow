# Contributing

Thank you for your interest in contributing to `skill-pr-review-workflow`.

## Development setup

```bash
git clone https://github.com/metyatech/skill-pr-review-workflow.git
cd skill-pr-review-workflow
```

## Submitting changes

1. Fork the repository and create a feature branch.
2. Add or update tests for any changed behavior.
3. Open a pull request with a clear description of the change.

## Scope

This skill covers **PR review feedback loops and GitHub notification handling**. Please keep PRs scoped to review workflow and GitHub integration concerns.
